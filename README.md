Student Management System
Ek simple aur efficient Python Console Application jo students ke academic records ko manage karne ke liye design ki gayi hai. Yeh project CRUD (Create, Read, Update, Delete) operations ko support karta hai aur data ko JSON file format mein store karta hai.

Features
Add Student: Naye student ka record (Name aur Grade) add karna.

Modify Student Score: Existing student ke grade ko update karna.

Remove Student: Kisi student ka record database se delete karna.

Search Student: Specific student ke record ko search karna aur tabular format mein dekhna.

Show All Records: Saare students ka data ek organized table mein dekhna.

Data Persistence: Data ko student_records.json file mein auto-save kiya jata hai.

Error Handling: Invalid input ko handle karne ke liye proper exception handling ka use kiya gaya hai.

Technology Stack
Language: Python

Storage: JSON (Local File)

How to Run
Python 3.x aapke system mein installed hona chahiye.

Repository ko clone karein ya code ko download karein.

Terminal ya Command Prompt mein file location par jayein.

Niche di gayi command run karein:

Bash
python main.py
Project Structure
main.py: Main application logic aur menu system.

student_records.json: Data storage file
