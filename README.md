# Password Generator & Strength Checker

A secure, cryptography-focused Python application that generates robust, customizable passwords and evaluates their complexity rating.

## Features
- **Well-Structured:** Logic is neatly organized into reusable functions for maintainability.
- **Accurate & Reliable:** Enforces strong password policies across lowercase, uppercase, digits, and special characters.
- **Security-Focused:** Utilizes Python's cryptographically secure `secrets` module instead of standard pseudo-random generators.
- **Clear Reporting:** Instantly checks and outputs the password's complexity rating (e.g., Excellent, Good).

## Technology Used
- Python 3.x
- Built-in Modules: `secrets`, `string`
