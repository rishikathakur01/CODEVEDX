# Weather Application using API Integration

Weather Application focuses on fetching real-time weather data using APIs.

## Features

* **Well-Structured:** API requests, response parsing, and display logic are clearly separated into functions.
* **Accurate and Reliable:** Proper error handling for network failures and invalid API keys is included.
* **User-Focused:** Displays temperature, humidity, wind speed, and weather conditions.
* **Clearly Reported:** Weather data is presented in a user-friendly and structured format.

## Technologies Used

* Python 3.x
* `requests` library

## Installation & Setup (Reproducible)

### 1. Prerequisites
Make sure Python is installed on your system. Install the required library using:
```bash
pip install requests
```
2. API Key Configuration
Sign up on a weather API provider (e.g., OpenWeatherMap).

Generate your unique API key.

Replace the placeholder API key variable inside get_weather_data.py with your active key.

3. Running the Application
Run the script using your terminal:

```Bash
python get_weather_data.py
```
